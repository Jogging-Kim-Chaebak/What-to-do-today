# 오늘 뭐할까
## 목표
- 노는 약속 계획 잡아주는 웹서비스

## 구현사항
- 낱말 퍼즐처럼 어디서 어떻게 뭐하고 놀지를 다수의 사용자가 선택해서 계획을 짤 수있다.
- 한 줄의 계획을 낱말로 완성할수 있다.

## 기술스택
- Spring Boot 사용
- 빌드 도구 maven 사용
- REST API 사용
- JPA
- DBMS : MariaDB
